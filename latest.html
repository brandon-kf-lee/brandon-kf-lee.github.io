Full title: A Full System Co-Simulation Platform for Evaluating Edge Machine Learning Inference Using Compute-in-Memory.

Full System Co-Simulation Platform

A full-system co-simulation framework for evaluating embedded machine learning inference on compute-in-memory (CIM) accelerators. It integrates a RISC-V embedded Linux guest in QEMU with a SystemC/TLM CIM memory model via MMIO, DMA, and interrupts. Using MNIST inference, CIM offload reduced CPU instruction count by 88.6% and estimated total system dynamic energy by 84.0%.
